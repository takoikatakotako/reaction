# 学習問題の管理

## 概要

学習問題は管理画面から追加・編集・削除・並び替えができます。
変更後は「データ更新」を実行して S3 に反映する必要があります。

## 管理画面の操作

### 問題一覧を確認する

管理画面のナビゲーションから「学習問題」をクリックすると、問題一覧が表示されます。
各問題には `#表示順 - ID` が表示され、最初の問題画像がプレビューされます。

### 問題を追加する

1. 問題一覧ページで「追加」をクリック
2. 以下の項目を入力:
   - **表示順**: 問題の並び順（数値）
   - **問題画像**: 問題の画像をアップロード（複数可）
   - **解答画像**: 解答の画像をアップロード（複数可）
   - **参考文献**: 参考文献のURLを入力（複数可）
3. 「追加」ボタンをクリック

### 問題を編集する

1. 問題一覧から編集したい問題をクリック
2. 各項目を編集
3. 「編集」ボタンをクリック

### 問題を削除する

1. 問題一覧から削除したい問題をクリック
2. 「削除」ボタンをクリック

### 表示順を変更する

問題の表示順は `order` フィールドで制御されます。
編集画面の「表示順」に数値を入力して保存してください。
数値が小さい順に表示されます。同じ数値の場合の順序は不定です。

## データ更新（S3エクスポート）

管理画面で問題データを変更しただけでは、iOS/Androidアプリには反映されません。
変更後は必ず以下の手順でデータを更新してください。

1. ナビゲーションの「データ更新」をクリック
2. 「更新」ボタンをクリック
3. 完了メッセージが表示されれば成功

この操作により、DynamoDB のデータが S3 に JSON として書き出され、CloudFront のキャッシュが無効化されます。
反応機構と学習問題の両方が更新されます。

## データの流れ

```
管理画面 → API → DynamoDB（マスターデータ）
                       ↓ データ更新
                      S3（JSON）→ CloudFront → iOS/Androidアプリ
```

## 画像について

- アップロードされた画像は S3 の `resource/image/` 配下に `{UUID}.png` として保存されます
- 画像は CloudFront 経由で配信されます
